# Personal Portfolio Website

A responsive, minimalist personal portfolio website built to showcase my programming projects, engineering background, and technical certifications.

## Live Demo
View the live site here: https://vj25ecb1a54-rgb.github.io/personal_portfolio/#contact

---

## Features
*   **Typing Animation:** Includes a JavaScript-driven typing effect on the introduction section.
*   **Project Filter:** Allows visitors to sort and filter projects dynamically by category.
*   **Dark / Light Mode Toggle:** Supports dynamic theme switching using custom CSS variables.
*   **Responsive Design:** Fully adjusted for mobile, tablet, and desktop viewports.
*   **Asset Access:** Fully configured download paths for my resume and direct contact links.

---

## Technical Stack
*   **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
*   **Libraries & Focus Areas:** Python libraries integrated into project showcase modules (Pandas, NumPy, Scikit-learn, TensorFlow, PyTorch)
*   **Hosting:** Configured for static site deployment via GitHub Pages or Netlify
